# Buttons
Use the button component to help users carry out an action like starting an application or saving their information.

Write button text in sentence case, describing the action it performs. For example:
* __'Search'__ to search the site
* __'Back to top'__ to return to the top of the page
* __'Report a benefit thief'__ a call to action indicating the start of a form
* __'Cancel'__ to stop the current task

:::warning Think before you use
Consider whether you should use a link instead of a button. For example, to provide access to a document such as a PDF, use a link.
:::

### Default button
The default button style that uses the Government's signature red colour.
This type of button is rarely used with the Primary button being preferred.

### Primary button
Use a primary button for positive actions, such as a continue button, as users respond better to green buttons.
If you are linking to a form, consider a [Call to action](#call-to-action-button) primary button.

### Call to action button
* The default option is rarely used
* Use the primary option for online forms
* Use the secondary option for a request to service and if you have multiple call to actions secondary to the main form
