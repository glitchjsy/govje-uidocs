# Basic Footer
The basic footer component is used at the bottom of every page.

### Regular footer
This is the footer used on gov.je, blog.gov.je, one.gov.je and more.
You should use this footer, along with an [our sites section](/our-sites), when possible.

:::warning NOTICE
The home page of gov.je uses a slightly modified version of this footer. See [Gov.je home page footer](#govje-home-page-footer) below.
:::
