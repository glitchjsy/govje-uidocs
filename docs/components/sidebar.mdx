# Sidebar & Side Panels
A sidebar is shown on the right of the page. It can have multiple **panels**. Only one sidebar can exist per page.

To create a sidebar in HTML:
```html
<div class="govje-sidebar">
    <div id="navigator">
        ...side panels here
    </div>
</div>
```
or React:
```jsx
<Sidebar>
    ...sidepanels here
</Sidebar>
```
